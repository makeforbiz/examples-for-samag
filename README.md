## Как использовать

1. Клонируйте репозиторий: `git clone https://github.com/makeforbiz/examples-for-samag.git`
2. Откройте соответствующий номер журнала «Системный администратор»;
3. Выполняйте команды из статьи, используя файлы из репозитория.
